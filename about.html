I’m Mark Jayson Lanuzo. I work on small, focused web tools—things that do one job well, load fast, and stay easy to maintain. I care about clear UI, sensible defaults, and color workflows that fit real design and development work.

This project

Color Scale Generator turns a single hex color into an 11-step scale (from 950 to 50), lets you manage multiple swatches, and exports CSS custom properties in HEX, RGB, or HSL—ready to paste into a stylesheet or token file.

The app is open source under the MIT License. Source code and updates live on GitHub: github.com/SonJaeMark/color-scale-generator.
